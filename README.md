# Smart Expense Tracker System

Smart Expense Tracker System is an expense management application designed to help users track daily expenses, manage budgets, and analyze spending patterns efficiently.

## Features
- Expense Management
- Budget Tracking
- Expense Categorization
- Monthly Spending Analysis
- Transaction History
- Dashboard-Based Analytics
- Secure REST APIs

## Tech Stack
### Backend
- Java
- Spring Boot
- Spring Framework
- REST APIs

### Frontend
- React.js
- HTML
- CSS
- JavaScript

### Database
- MySQL

### Tools & Platforms
- Git
- GitHub
- Postman

## System Functionalities
- Add, update, and delete expenses
- Categorize expenses for better tracking
- Monitor monthly budgets
- View transaction history
- Store and retrieve expense data efficiently
